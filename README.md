# Consult Marketing Pages

This package includes two separate Next.js landing pages:

- `/surgery-consult`
- `/general-consult`

Before you deploy, replace these placeholders:
- `REPLACE_WITH_SURGERY_FORM_ID`
- `REPLACE_WITH_GENERAL_FORM_ID`

Use your Formspree endpoints in this format:
- `https://formspree.io/f/your_surgery_id`
- `https://formspree.io/f/your_general_id`

Run locally:
1. npm install
2. npm run dev
3. Open http://localhost:3000
